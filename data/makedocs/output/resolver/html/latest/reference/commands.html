Command Reference

compile

With the compile command it's possible to compile the project including its dependencies.

Usage

resolver [options] compile

Arguments and Options

-w [path] --working-dir [path] Makes it possible to specify the working directory to execute in.
-s --silent Disables all log messages
-d --dependencies Only compiles the dependencies of the project.
-p --project Only compiles the project and leaves the dependencies as is.

Exit Code

This command will exit with the code 0 when it succeeds; 1 in case of an error.

generate

Generates IDE project files depending on your project configuration.

Usage

resolver [options] generate [ide]

Arguments and Options

-w [path] --working-dir [path] Makes it possible to specify the working directory to execute in.
-s --silent Disables all log messages
[ide] The IDE to generate the project files for. Current supported IDE's are:
  • vs2010

Exit Code

This command will exit with the code 0 when it succeeds; 1 in case of an error.

update

Updates the dependencies to the version configured in your config file. When the dependencies are not installed yet, Resolver will install them.

Usage

resolver [options] update

Arguments and Options

-w [path] --working-dir [path] Makes it possible to specify the working directory to execute in.
-s --silent Disables all log messages

Exit Code

This command will exit with the code 0 when it succeeds; 1 in case of an error.
